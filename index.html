ThunderHack Configuration Guide

Hi there! I'm pretty sure you're looking at this file for a guide on how to create configs? Well, let's go over these two giant rule sections: Legit and Unlegit.

Legit Rules

  • Do not go over 3.00 reach or it will be detected by many ACs.
  • You must always have rotations on; if not, you will be banned very easily.
  • Do not use modules that are blatant. Always read what they're actually about and understand which config is legit and which is not.
  • All renders are legit as they don't interact with your movements or clicks, just rendering a visual effect.
  • Check any other modules if they are legit, and configure them before usage.
  • Your options in hand-to-hand combat, not crystaling, are triggerbot and killaura with rotations.
  • Module sections containing both unlegit and legit modules: combat, misc, movement, player.
  • Always configure modules before use as there is no 100% undetected unconfigured module.

Unlegit Rules

  • Same as legit rules, but you can do the opposite of what it tells you not to do.
  • Include legit modules too, as at this point it's just unlegit.
  • Configure 100% like before as normal configs can get you banned without being configured.

Legit Modules

  • triggerbot (configure the range to 3)
  • legithelper (requires configuration)
  • elytrarecast (requires configuration)
  • hitboxes (best for entities like crystals or players; requires configuration)
  • auto armor (configure to change armor when standing still; add 1 delay)
  • fake player (no configuration needed unless desired)
  • auto totem (requires intense configuration; not sure if legit)
  • inventory cleaner (configure by adding items for the cleaner to clean using @invcleaner)
  • teams (no module, use middleclick to friend entity; easy configuration)
  • Discover more by configuring modules to a legit point.

Configurations for Legit

  • Use your brain; ensure all configurations seem legit.
  • Never go over 3.00 reach.
  • Never use wall reach.
  • Always use rotations.
  • Use middleclick for team/friend by making it friend on entity.
  • Use renders as much as possible, but avoid those that reduce framerates.
  • Always use 1920x1080 resolution for HUD editing.
  • Avoid using less common modules unless configured properly.
  • Add unlegit modules depending on server's weak spots.
  • Perfect the delay settings.
  • No legit autocrystal aura; only a legit macro for it.

Configurations for Unlegit

Not needed because it's the same as legit but configured to be unlegit or blatant. Configure depending on ACs' weak spots or for specific servers.

Most Important Stuff

  • Always list the bindings in your configuration posts.
  • Understand how to use norender and never enable the items part as it deletes items.
  • For basehunting, add storage ESP, tunnel ESP, and search.
